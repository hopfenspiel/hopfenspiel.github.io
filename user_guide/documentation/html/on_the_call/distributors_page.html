The Distributors Page

The On The Call tab starts with the Distributors page, which is broken up into four main sections:

  1. Search and Navigation
  2. Favorited Distributors
  3. Nearby Distributors
  4. All Distributors

We’ll go over each of these in detail.

Search and Navigation

The search bar allows you to narrow down the list of distributors displayed. To use this feature, simply enter the search bar and start typing the name of the desired distributor:

The list of distributors shown in both the Distributors section and the Favorites section will refine based on the text entered in the search bar; the Favorites section will go blank if the search text does not match any favorited distributors.

Name

This is the default format for displaying distributors; that is, in alphabetical order by distributor name.
To split up distributors by state, the user must press the State button.

State

To break distributor results up by state, press the State button. This will cause an alphabetically arranged state list to appear, with the number of distributors for each state displayed along with the state name.

Choose the desired state, and off you go! You’ll now see a list of all distributors serving that state.

NOTE: It’s not possible to search by distributor names in the “by State” distributor view, and Favorites will only appear if they belong within that state.

Favorites

Favorited distributors are distributors you’d like to easily access every time you log in to the app, and the Favorites section is where these favorited distributors are displayed.

As mentioned in the screenshot, all favorited distributors are shown with a full mason jar next to their name (mmm…tasty).

How to Favorite a Distributor

To “favorite” a distributor, either search or scroll for that distributor in the Distributors list, and press the empty mason jar next to the desired distributor:

Once pressed, the empty jar will be replaced with a full glass of goodness, and the distributor will now be included in your list of Favorites.

Nearby

This section shows all the distributors that serve your nearby area (nearby is based on the GPS location of your iPad).

We’re in Petaluma, so we’re seeing Columbia Santa Rosa – whaddy’a expect?!

Distributors

This section shows all distributors that, well, distribute our beer. Organized alphabetically, the location (city, state) and number of accounts for that distributor are displayed along with the distributor name.

Here you can choose a distributor to “favorite” (see the Favorites section above); you can also select a distributor and see all the accounts for which that distributor is responsible (more on that in the next section).
