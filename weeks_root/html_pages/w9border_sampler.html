Week 9 Border Sampler

The border exaxmples shown here were adapted from the border sampler source page. I used the outlines of the borders and changed things to make them my own.

This text is surrounded by a solid border. The CSS - border: 10px solid #6633FF;

This is a dotted border. The CSS - border: 8px dotted #A3F7BF;

This is a dashed border. The CSS - 6px dashed #4B6289

This is a medium double border. The CSS - border:medium double #60316E;

This is a large outset border. The CSS - border:20px outset #A3F7BF;

This is a thick inset border. Doesn't seem that thick! The CSS - border:thick inset #A3F7BF;

This is a groove border. I like the alternating color pattern.
I wonder how the browser knows which alternate color to use?
The CSS - border:15px groove #F6C90E;

This is a ridge border, which seems to be the opposite of a groove border.. The CSS - border:15x ridge #F6C90E;

This text is surrounded by a variety of border styles. The CSS - border-top:6px dashed #FF0066; border-right: 3px dotted #6633FF; border-bottom:5px ridge #3399CC; border-left:4px double #00CC33;
